# @emdzej/ediabasx-protocol-doip

DoIP / **HSFZ** (ISO 13400) primitives for [EdiabasX](https://github.com/emdzej/ediabasx) — used by the ENET transport for modern BMW vehicles.

> **Status: work in progress.** The package is a placeholder; the DoIP message types, routing-activation handshake, and HSFZ framing are being implemented as the ENET transport matures.

## Install

```bash
pnpm add @emdzej/ediabasx-protocol-doip
```
